QA Framework - Windows Installation Guide

Step 1: Python

  • Download and install from https://www.python.org/downloads/
    • Check Add Python to PATH during installation

Step 2: Download Programs

  • VS Code: https://code.visualstudio.com/
  • GitHub Desktop: https://desktop.github.com/
    • Git is installed automatically
  • Docker Desktop (AMD): https://www.docker.com/products/docker-desktop/
    • Use WSL 2 backend when prompted
    • After install, open a terminal and run: wsl --update

Step 3: Clone Repository

  • Open GitHub Desktop and sign in with your GitHub account
  • Clone the qa_framework repository to C:\Coding\qa_framework
  • Open the folder in VS Code: File → Open Folder

Step 4: VS Code Extensions

These are the recommended extensions.
  • Open Extensions panel (Ctrl+Shift+X) and install:
    • Python (Microsoft)
    • Pylance (Microsoft)
    • Ruff (Astral Software)
    • Playwright Test for VS Code (Microsoft)
    • Docker (Microsoft)
    • Makefile Tools (Microsoft)
    • GitLens (GitKraken)
    • Windsurf (Windsurf)
      • You will need to make a free account as part of the extension installation

Step 5: Install Dependencies

  • Run python install.py
    • Installs: uv + dependencies, Scoop, Java, Make, Allure
  • Open a new terminal. It will automatically put you into the virtual environment.
    • If it does not, follow these steps.
    • Press Ctrl+Shift+P
    • Python: Select Interpreter
    • Select the one with .venv in the path
    • Open a new terminal, it should now automatically put you into the virtual environment

Step 6: Environment Variables

  • In the project root, create .env file if it's missing.
  • Open .env and fill in all values
  • Use .env.example as a reference for required keys

Step 7: Verify Installation

  • Run this command, expecting all tests to pass: make test
