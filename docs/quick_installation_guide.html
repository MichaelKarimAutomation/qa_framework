QA Framework - Windows Installation Guide

Step 1: Python

  • Download and install from https://www.python.org/downloads/
  • Check Add Python to PATH during installation

Step 2: Download Programs

  • VS Code: https://code.visualstudio.com/
  • GitHub Desktop: https://desktop.github.com/
    • Git is installed automatically
  • Docker Desktop (AMD): https://www.docker.com/products/docker-desktop/
    • Use WSL 2 backend when prompted
    • After install, open a terminal and run: wsl --update

Step 3: Clone Repository

  • Open GitHub Desktop and sign in with your GitHub account
  • Clone the qa_framework repository to C:\Coding\qa_framework
  • Open the folder in VS Code: File → Open Folder

Step 4: VS Code Setup

  • Open Extensions panel (Ctrl+Shift+X) and install:
    • Python (Microsoft)
    • Pylance (Microsoft)
    • Ruff (Astral Software)
    • Playwright Test for VS Code (Microsoft)
    • Docker (Microsoft)
    • Makefile Tools (Microsoft)
    • GitLens (GitKraken)
    • Windsurf (Windsurf)
      • You will need to make a free account as part of the extension installation

Step 5: Install uv

  • Open a terminal in VS Code (Terminal → New Terminal) and run: 
    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
  • Open Edit environment variables for your account → Path → New and add: 
    C:\Users\YourUsername\.local\bin
  • Close and reopen the terminal, then verify: 
    uv --version

Step 6: Scoop, Java, Make, and Allure

  • In PowerShell terminal, run: 
    powershell -ExecutionPolicy ByPass -c "irm get.scoop.sh | iex"
  • Open Edit environment variables for your account → Path → New and add: 
    C:\Users\YourUsername\scoop\shims
  • Close and reopen the terminal, then run: 
    scoop bucket add java
scoop install temurin-lts-jdk make allure
  • Verify: 
    allure --version
make --version

Step 7: Environment Variables

  • In the project root, copy .env.example to .env: 
    copy .env.example .env
  • Open .env and fill in all values

Step 8: Virtual Environment and Dependencies

  • In the VS Code terminal, run: 
    uv venv
.venv\Scripts\activate
uv add pytest pytest-xdist pytest-rerunfailures playwright pytest-playwright httpx allure-pytest faker factory-boy sqlalchemy psycopg2-binary testcontainers pytest-dotenv ruff pytest-httpserver locust jsonschema
playwright install

Step 9: Select Python Interpreter

  • Press Ctrl+Shift+P → type Python: Select Interpreter → select the one with .venv in the path

Step 10: Verify Installation

  • Run all tests: 
    make test
  • All tests should pass.
